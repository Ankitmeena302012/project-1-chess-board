# Chess Board

A simple chess board built with HTML and CSS.

## Structure

- **index.html** - Contains the chess board with 64 squares (8x8 grid) and Unicode chess pieces in starting position
- **style.css** - Styles the board with alternating black and white squares

## How It Works

- The board is a 640px × 640px grid made of 64 div elements
- Each square is 80px × 80px
- `box1` class creates dark squares (black background, white text)
- `box2` class creates light squares (white background, black text)
- Chess pieces are rendered using Unicode characters (♔♕♖♗♘♙ for white, ♚♛♜♝♞♟ for black)

## To View

Open `index.html` in any web browser.
